 ATM Machine Simulation (Java)

A simple console-based ATM Machine simulation built using Java.
📌 Features
- PIN Authentication
- Check Balance
- Deposit Money
- Withdraw Money
- Menu-driven system
 🛠 Technologies Used
- Java
- Scanner class
- Switch case
- Loops
 ▶️ How to Run
Compile:
javac ATM.java
Run:
java ATM
